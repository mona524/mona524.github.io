web.py url路由配置和分发

发表于 2018-04-28   |   分类于 python

一、简单URL规则：

import web
urls = (
    '/index','Index',
    '/hello/\d+', 'Hello',
    '/hey/(.*)','Hey',
)
class Index:
    def GET(self):
        return "index page"
class Hello:
    def GET(self):
        return 'hello world'
class Hey:
    def GET(self,args):
        return 'hey'+ args
if __name__ == "__main__":
    app = web.application(urls, globals())
    app.run()

1、URL完全匹配/index 访问路径http://127.0.0.1:8080/index 处理函数Index

2、URL模糊匹配 /hello/\d+ 访问路径 http://127.0.0.1:8080/hello/9 处理函数Hello

3、URL带组匹配 hey/(.*) 访问路径http://127.0.0.1:8080/hey/mona 处理函数Hey

二、路由分发：
当路由较多时，可以将URL单独写一个文件，取名urls.py,然后在index.py中引用

index.py 文件，记住一定要导入处理函数和URL

import web
from urls import mona_url
from login import Login
from login import Hello
from login import Hey
from login import Index
if __name__ == "__main__":
    app = web.application(mona_url, globals())
    app.run()

urls.py 文件

# !/usr/bin/env python
# -*- coding:utf-8 -*- 
# Author: mona
# time: 2018/4/28
mona_url = (
    '/index', 'Index',
    '/hello/\d+', 'Hello',
    '/hey/(.*)', 'Hey',
    '/login','Login',
)

login.py 文件

# !/usr/bin/env python
# -*- coding:utf-8 -*- 
# Author: mona
# time: 2018/4/28
class Login:
    def GET(self):
        return "login page"
class Index:
    def GET(self):
        return "index page"
class Hello:
    def GET(self):
        return 'hello world'
class Hey:
    def GET(self,args):
        return 'hey'+ args

三、两个或多个参数的URL
url 配置'/hey/(.*)/(.*)', 'Hey',

处理函数：

class Hey:
def GET(self,args,name):
    return 'hey'+ args+name

简单总结：


  • 在web.py中，url后有无”/“是不一样的

  • 静态文件不需要写映射，只需要放在static目录中

  • 使用web.seeother和web.redirect完成跳转。前者更加友好，返回303状态码，而后者返回301

  • 返回xml或者json等格式数据时，需要在web.header中设置数据的格式

  • 通过web.input/web.data获取url请求参数
